# Mobile UX Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the app comfortable for primary mobile use — bottom navigation, simplified header, Add Book FAB, and tap-target/layout fixes — without touching the desktop experience.

**Architecture:** Four independent changes: (1) a new `BottomNav` client component added to the root layout, (2) a one-line header tweak to hide user info on mobile, (3) a FAB added to the books list page, and (4) three small CSS/markup fixes. All mobile behaviour is gated behind Tailwind's `md:` breakpoint (< 768px shows mobile UI, ≥ 768px is unchanged).

**Tech Stack:** Next.js 15 App Router, Tailwind CSS v4, lucide-react icons, `usePathname` from `next/navigation`.

## Task 1: BottomNav component + layout wiring

**Files:**
- Create: `components/bottom-nav.tsx`
- Modify: `app/layout.tsx`

No unit-testable logic — the only logic is active-route detection which depends on Next.js `usePathname`. Verify with typecheck.

- [ ] **Step 1: Create `components/bottom-nav.tsx`**

```tsx
'use client'

import Link from 'next/link'
import { usePathname } from 'next/navigation'
import { Home, BookOpen, Users } from 'lucide-react'

const NAV_ITEMS = [
  { label: 'Home',      href: '/',           icon: Home,     exact: true  },
  { label: 'Books',     href: '/books',       icon: BookOpen, exact: false },
  { label: 'Borrowers', href: '/borrowers',   icon: Users,    exact: false },
]

export function BottomNav() {
  const pathname = usePathname()

  return (
    <nav className="fixed bottom-0 left-0 right-0 z-50 flex h-14 border-t bg-background md:hidden">
      {NAV_ITEMS.map(({ label, href, icon: Icon, exact }) => {
        const active = exact ? pathname === href : pathname.startsWith(href)
        return (
          <Link
            key={href}
            href={href}
            className={`flex flex-1 flex-col items-center justify-center gap-0.5 text-xs ${
              active ? 'text-primary' : 'text-muted-foreground'
            }`}
          >
            <Icon size={20} />
            <span>{label}</span>
          </Link>
        )
      })}
    </nav>
  )
}
```
