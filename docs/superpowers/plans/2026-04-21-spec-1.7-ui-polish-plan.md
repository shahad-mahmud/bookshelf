# UI Polish Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a navigation progress bar and success toasts to make the app feel responsive and give users confirmation that actions worked.

**Architecture:** Two independent changes — (1) install `nextjs-toploader` and drop one component into the root layout for instant navigation feedback, (2) import `toast` from `sonner` (already wired) and call `toast.success()` in the three existing `wasPendingRef` success effects.

**Tech Stack:** Next.js 15 App Router, `nextjs-toploader`, `sonner` (already installed).

- [ ] **Step 2: Update `app/layout.tsx`**

Current file:
```tsx
import type { Metadata } from 'next'
import { Toaster } from '@/components/ui/sonner'
import { BottomNav } from '@/components/bottom-nav'
import './globals.css'

export const metadata: Metadata = {
  title: 'Bookshelf',
  description: 'Track books you own, lend, and want.',
}

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en" suppressHydrationWarning>
      <body className="min-h-svh bg-background font-sans antialiased pb-16 md:pb-0">
        {children}
        <Toaster />
        <BottomNav />
      </body>
    </html>
  )
}
```

Replace with:
```tsx
import type { Metadata } from 'next'
import NextTopLoader from 'nextjs-toploader'
import { Toaster } from '@/components/ui/sonner'
import { BottomNav } from '@/components/bottom-nav'
import './globals.css'

export const metadata: Metadata = {
  title: 'Bookshelf',
  description: 'Track books you own, lend, and want.',
}

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en" suppressHydrationWarning>
      <body className="min-h-svh bg-background font-sans antialiased pb-16 md:pb-0">
        <NextTopLoader color="#2299DD" showSpinner={false} />
        {children}
        <Toaster />
        <BottomNav />
      </body>
    </html>
  )
}
```
